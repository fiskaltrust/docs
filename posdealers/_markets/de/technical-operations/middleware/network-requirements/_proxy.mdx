### Swissbit Cloud & Deutsche Fiskal TSE

The SCU package for these TSE types supports the following four optional SCU parameters:

* ProxyServer
* ProxyPort
* ProxyUsername
* ProxyPassword

More information is available at the [Swissbit Cloud documentation](../../../../../../poscreators/middleware-doc/middleware-de-kassensichv/operation-modes/scu/swissbit-cloud.md).
