# Android Intent Integration

On Android, the POS System API can be invoked through an intent-based integration approach for communication with fiskaltrust components.

For end-to-end examples and integration details, see the [POS System API Development Kit](https://github.com/fiskaltrust/possystemapi-devkit/blob/main/README.MD).
