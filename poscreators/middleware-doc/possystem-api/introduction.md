# Introduction

The **POS System API** provides a stable interface for POS systems to communicate with the fiskaltrust Middleware.

For implementation samples and request/response examples, see the [POS System API Development Kit](https://github.com/fiskaltrust/possystemapi-devkit/blob/main/README.MD).
