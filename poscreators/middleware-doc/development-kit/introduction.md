# Introduction

This page serves as your starting point for integrating with the fiskaltrust POS System API using the official developer resources provided by **fiskaltrust**.

The full technical documentation, setup instructions, and implementation details are available in the official GitHub repository: [fiskaltrust POS System API - Development Kit](https://github.com/fiskaltrust/possystemapi-devkit/blob/main/README.MD)

## Getting Started

To begin:

1.	Open the official DevKit README.
2.	Follow the setup instructions.
3.	Review the API structure and examples.
4.	Implement and test your integration.

For complete and up-to-date technical documentation, always refer to the GitHub repository linked above.
